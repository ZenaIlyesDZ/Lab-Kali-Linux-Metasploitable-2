#Installation
For this project, we need 2 Virtual Machine : Metasploitable 2 and Kali Linux
### Metasploitable 2
Metasploitable 2 is a vulnerable machine based on Ubuntu, it was created by Rapid7 to learn basics of pentesting
### Kali Linux
Kali Linux is a Linux distribution specialy made for pentesting, it is the most used OS in pentesting
